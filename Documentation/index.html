Getting Started - top

Installation

As this is a purely HTML template, installation couldn't be any easier. To install the template to your website simply upload the contents of the HTML folder to your webserver of choice. This is normally the public_html folder or httpdocs folder. If you aren't sure which folder you need to upload the website to then your webhost should be able to offer you assistance with this.

There should be a number of folders uploaded to your server for the website to display correctly.

Firstly, the index.html should be uploaded.

After this there should be the following folders:

  • CSS stylesheets
  • Javascript files
  • Images folder
  • Fonts folder (optional unless you're going to use Bootstrap's Glyphicons!)

Modifying Files

If you'd like to modify any of the files, simply open them up in an editor. If you don't have an editor already, you can download one if you'd like to. Below are two of the popular editors.

  • Notepad++
  • Sublime Text

Once you've download en editor, simply open up the index.html file. That's the main file which is basically the whole theme. You can change anything you want that suits you in this file.

Opening up the style.css stylesheet will show you the styling of the theme. You can pretty much customize the color, buttons or pictures to your liking. The style.min is the compressed version of the style.css.

Bootstrap Framework
